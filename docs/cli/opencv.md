# opencv — Computer Vision (OpenCV 4.13)

Computer vision (OpenCV 4.13, via python3+cv2): QC (edges/compare/count/template_match), logistics & retail (read_qr/dominant_colors), security (detect_faces/motion/blur_faces), doc intake (document_scan/threshold), imaging (denoise), and capture (extract_frames/stitch). Run `opencv_doctor` first.

How to read this page:

- **Tool `name`** is the exact string for `tools/call`; **Workflow `name`** is `wf__<id>`.
- **Consent = yes** means the call changes state — over MCP it triggers an elicitation (or launch the server with `-y` to pre-approve). Consent = no calls are read-only.
- **LLM = mid/high** means the call sends content to a configured language model; `—` calls are fully local and free.
- If a call errors with "tool category disabled", enable it: `enable_category {"category":"opencv"}` (or launch with `--enable opencv`).
