# mysql ‚Äî MySQL/MariaDB

How to read this page:

- **Tool `name`** is the exact string for `tools/call`; **Workflow `name`** is `wf__<id>`.
- **Consent = yes** means the call changes state ‚Äî over MCP it triggers an elicitation (or launch the server with `-y` to pre-approve). Consent = no calls are read-only.
- **LLM = mid/high** means the call sends content to a configured language model; `‚Äî` calls are fully local and free.
- If a call errors with "tool category disabled", enable it: `enable_category {"category":"mysql"}` (or launch with `--enable mysql`).
